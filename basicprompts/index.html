Estructura de cada prompt — framework C.R.E.I.
Contexto
Define la situación, empresa o audiencia. Cuanto más específico, mejor resultado.
Referencia
El tono, estilo o formato que debe tener la respuesta.
Evaluación
Los criterios mínimos que debe cumplir una buena respuesta.
Iteración
Una variación del prompt para profundizar o cambiar el enfoque.
¿Cómo usar estos ejercicios? Revisa la pestaña Estructura para entender el prompt, luego lee los Tips antes de ejecutarlo. Cuando estés listo, ve a Copiar prompt para seleccionar el texto y pegarlo en tu LLM favorito.
