Idea clave: Un Proyecto no te hace más inteligente — te quita fricción. En lugar de reexplicar el contexto cada vez que abres ChatGPT, el Proyecto lo recuerda por ti: tus instrucciones, tus archivos y el historial de trabajo, todo en un solo lugar.

¿Qué es un Proyecto?
Espacio de trabajo inteligente
Un contenedor que agrupa chats, archivos e instrucciones relacionados a un mismo esfuerzo continuo — sin tener que reexplicar el contexto en cada conversación.
Memoria y coherencia
Recuerda chats y archivos del proyecto entre sesiones. Las instrucciones del proyecto se aplican a todas las conversaciones y reemplazan las instrucciones globales.
Colaboración por roles
Puedes invitar personas con nivel Chat (solo conversan) o Editar (modifican instrucciones y archivos). Disponible desde el plan Free.
Herramientas integradas
Dentro de un Proyecto puedes usar búsqueda web, canvas, generación de imágenes, modo de estudio y —según plan— agente e investigación profunda.
La analogía que lo explica todo
Piénsalo como una carpeta de trabajo con cerebro propio
Instrucciones El "contrato" con ChatGPT: cómo debe responder, qué tono usar, qué formato entregar. Se aplican automáticamente a todos los chats del proyecto.
Fuentes Los archivos, textos y enlaces que definen la base de conocimiento del proyecto. PDFs, hojas de cálculo, documentos, canales de Slack, carpetas de Drive.
Chats Las conversaciones dentro del proyecto. Todas heredan las instrucciones y fuentes. Puedes ramificarlas para explorar variantes sin perder el hilo original.
Memoria El proyecto recuerda lo que ocurrió en sesiones anteriores. Puedes guardar una respuesta útil como fuente del proyecto para reutilizarla después.

Tres problemas que resuelve
Continuidad
En lugar de que cada chat sea una isla, el Proyecto mantiene un núcleo de contexto que reduce el re-trabajo en cada sesión.
Consistencia
Las instrucciones del proyecto prevalecen sobre las instrucciones globales, asegurando el mismo tono y formato en todas las conversaciones.
Reusabilidad
Puedes guardar respuestas útiles como fuentes del proyecto: resúmenes, decisiones, análisis — disponibles en futuras sesiones sin buscar en chats pasados.
