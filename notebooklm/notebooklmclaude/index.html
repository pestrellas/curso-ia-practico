Paso 01 — NotebookLM / Discover Sources
Recolectar fuentes técnicas sobre energía en LATAM
Usas la función Discover Sources de NotebookLM para enviar al agente a navegar más de 100 sitios simultáneamente y traer reportes técnicos del BID, CEPAL, FMI, PwC, KPMG y otros organismos de alta autoridad.
NotebookLM
Abre NotebookLM y crea un cuaderno nuevo
Ve a notebooklm.google.com, inicia sesión con tu cuenta de Google y haz clic en "Nuevo cuaderno". Dale un nombre como "Inteligencia Energética LATAM".
Activa "Discover Sources"
En el panel de fuentes (lado izquierdo), busca el botón Discover Sources o el ícono de búsqueda web. Este es el modo agéntico que navega la web.
Copia y pega el prompt de recolección
Usa el botón de abajo para seleccionar el prompt, cópialo con Ctrl+C y pégalo en el campo de Discover Sources. El agente buscará 15 fuentes técnicas verificadas.
Importa las fuentes al cuaderno
Cuando el agente termine, verás una lista de fuentes con justificación técnica. Selecciona las más relevantes y haz clic en Agregar al cuaderno. Espera que se procesen.
Criterio de selección: Prioriza fuentes con más de 15 páginas, metodología estadística y tablas de datos. Descarta artículos de opinión o noticias cortas — el prompt ya le indica esto al agente, pero conviene verificarlo.

Texto seleccionado — presiona Ctrl+C (Windows) o Cmd+C (Mac)
Paso 02 — NotebookLM / Chat del cuaderno
Extraer los datos cuantitativos de las fuentes
Con las fuentes ya importadas en tu cuaderno, cambias al chat de NotebookLM para pedirle que organice todos los datos numéricos relevantes. Este resultado es la materia prima que llevarás a Claude en el paso siguiente.
NotebookLM
Cambia al chat del cuaderno
Una vez que todas las fuentes estén procesadas (verás una palomita verde junto a cada una), haz clic en el área de chat a la derecha del cuaderno para comenzar una conversación.
Copia y pega el prompt de extracción
Pega el prompt de abajo en el chat. NotebookLM analizará todas las fuentes importadas y organizará los datos cuantitativos en bloques temáticos.
Copia el resultado completo
Cuando NotebookLM entregue la respuesta con los datos organizados por bloques, selecciona todo el texto y cópialo. Este contenido lo pegarás en el prompt de Claude en el paso 3.
Consejo: Si el resultado es muy extenso, pide a NotebookLM que lo divida: "Organiza los datos en bloques de: generación convencional, renovables, demanda digital y proyecciones 2030". Eso facilita el trabajo de Claude después.

Luego Ctrl+C / Cmd+C
Texto seleccionado — presiona Ctrl+C (Windows) o Cmd+C (Mac)
Paso 03 — Claude
Generar el dashboard interactivo
Abres Claude y pegas el prompt de dashboard junto con el resultado que copiaste de NotebookLM. Claude generará un artefacto interactivo con gráficos, KPIs y recomendaciones estratégicas basadas en los datos reales de las fuentes.
Claude
Abre Claude en claude.ai
Ve a claude.ai e inicia una nueva conversación. No necesitas ninguna configuración especial — el prompt ya lleva toda la instrucción.
Copia el prompt de dashboard
Selecciona el prompt con el botón de abajo. Al pegarlo en Claude, reemplaza el marcador :: Coloca aquí el resultado :: con el texto que copiaste de NotebookLM en el paso anterior.
Claude genera el artefacto automáticamente
Claude producirá un dashboard con gráficos interactivos de crecimiento, visualización de renovables vs. convencionales, niveles de adopción y KPIs. Revisa que los datos coincidan con tus fuentes.

Paso 04 — Publicar y compartir
Convertir el dashboard en un recurso público
Una vez que Claude entrega el dashboard, lo conviertes en un artefacto publicable con una URL propia. Puedes compartirlo como enlace o incrustarlo en Moodle, una presentación o cualquier página web.
Publicar
Pide a Claude que genere el artefacto
En la misma conversación de Claude, envía el siguiente mensaje exacto: "Genera un artefacto con este resultado". Claude empaquetará el dashboard como un componente interactivo independiente.
Publica el artefacto
En el panel del artefacto (lado derecho de Claude), haz clic en el botón Publicar o el ícono de compartir. Claude generará una URL pública con el formato claude.ai/public/artifacts/...
Comparte el enlace o incrústalo
Copia la URL para compartirla directamente. Para incrustar en Moodle o una web, usa el código iframe que Claude también provee al publicar.
Prompt exacto para este paso: Una vez que Claude entregue el dashboard, envía en la misma conversación:

"Genera un artefacto con este resultado"

Luego usa el botón de publicar en el panel del artefacto para obtener la URL y el código de incrustación.
Resultado final: Un dashboard interactivo con URL propia, incrustable en cualquier plataforma, basado en fuentes técnicas verificadas del BID, CEPAL, McKinsey y otros organismos de autoridad sobre el sector energético en América Latina.
